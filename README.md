# Tower of Hanoi

A classic puzzle game built as a single-page web application.

## Play

Move all disks from the first peg to the last peg. You can only move one disk at a time, and a larger disk cannot be placed on top of a smaller one.

**[Play the game →](https://yourusername.github.io/hanoi-tower/)**

## Features

- 3 difficulty levels (3, 4, or 5 disks)
- Move counter with optimal solution display
- Smooth animations
- Mobile-friendly design

## Setup

1. Clone this repository
2. Open `index.html` in your browser

Or host it on GitHub Pages — no build step required.
